# Spec 000: Spec-Driven LLM Wiki Architecture

This architecture is defined in detail by `../../full-spec.md`.

## Purpose

Provide a self-contained methodology for spec-driven development with persistent wiki memory, graph visualization, TypeScript tooling, and session handoffs.

## Current MVP Scope

- Agent instruction files.
- Prompt source files.
- Wiki/spec/graph skeleton.
- Deterministic TypeScript validation and graph generation.
- HTML graph rendering.
- Documentation and templates.

## Deferred Scope

- LLM semantic edge inference.
- Full query engine.
- Cross-session semantic contradiction detection.
- Dashboard UI beyond generated graph HTML.
