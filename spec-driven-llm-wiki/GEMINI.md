# Spec-Driven LLM Wiki - Gemini CLI

Use `AGENTS.md` as the canonical workflow contract.

Supported triggers:

- `spec <task>`
- `implement <spec-id>`
- `build graph`
- `lint`
- `status`
- `handoff`
- `query: <question>`

Read `spec/prompts/BEHAVIOUR.md` and `spec/prompts/INSTRUCTIONS.md` before executing a workflow.
