# Workflow Reference

## SPEC

Research parent project, write a numbered spec, update wiki memory, and log the work.

## IMPLEMENT

Read a spec, edit only scoped files, validate, update spec status, wiki, graph, and handoff.

## GRAPH

Build deterministic graph data from specs and wiki pages. Optional LLM inference is disabled by default.

## LINT

Validate specs, wikilinks, graph edges, and tool docs.

## HANDOFF

Capture current state so a future session can resume without re-discovery.
